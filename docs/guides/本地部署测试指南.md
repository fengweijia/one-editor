## 一、环境要求

| 环境 | 版本 | 说明 |
|:---|:---:|:---|
| Python | 3.11+ | 后端运行 |
| Node.js | 18+ | 可选，前端为纯静态 |
| 网络 | 可访问外网 | 需访问Jina Reader API |

---

## 二、快速启动

### 1. 克隆项目

```bash
git clone <oneeditor-repo>/OneEditor-main.git
cd OneEditor-main
```

### 2. 安装依赖

```bash
cd backend
pip3 install -r requirements.txt

# 或手动安装核心依赖
pip3 install fastapi uvicorn httpx requests trafilatura python-dotenv pydantic
```

### 3. 启动服务

```bash
cd backend
python3 -m uvicorn app:app --host 0.0.0.0 --port 8000
```

### 4. 访问

- 前端页面：http://localhost:8000/
- API文档：http://localhost:8000/docs

## 三、API接口测试

### 测试1：URL抓取

```bash
curl -X POST http://localhost:8000/ingest/url \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com"}'
```

**预期响应**：
```json
{
  "status": "ok",
  "data": {
    "platform": "web",
    "canonical_url": "https://example.com",
    "meta": {"title": "Example Domain"},
    "normalized_text": "...",
    "checksum": "..."
  }
}
```

## 四、飞书配置（如需使用）

### 配置步骤

1. 创建飞书多维表格应用
2. 获取 App ID 和 App Secret
3. 创建多张表格：
   - 观点库 (table_points)
   - 案例库 (table_cases)
   - 金句库 (table_quotes)
   - 结构库 (table_structures)

### 配置环境变量

在 `backend/src/config/settings.py` 中设置：
```python
feishu_settings = FeishuSettings(
    app_id="your_app_id",
    app_secret="your_app_secret",
    app_token="your_app_token",
    table_points="table_id_1",
    table_cases="table_id_2",
    table_quotes="table_id_3",
    table_structures="table_id_4"
)
```

## 六、常见问题

### Q1: 启动报错 "No module named 'fastapi'"

```bash
pip3 install fastapi uvicorn
```

### Q2: URL抓取失败

- 检查网络是否可访问外网
- 尝试使用Jina Reader（已内置）

### Q3: AI分析结果为空

- 这是正常现象，无API配置时使用fallback模式
- 配置LLM API后可以获得完整解析

### Q4: 页面显示异常

- 确保使用 Chrome/Edge/Firefox 最新版
- 检查网络是否可访问 Google Fonts
